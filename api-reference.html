API Reference

Complete REST API documentation for OmniMind24 platform.

Authentication

All API requests require authentication using Bearer tokens:

Authorization: Bearer YOUR_API_TOKEN

Campaigns

GET /api/campaigns

Retrieve all campaigns for the authenticated user.

POST /api/campaigns

Create a new campaign with COSTAR framework data.

GET /api/campaigns/:id

Retrieve a specific campaign by ID.

Content Generation

POST /api/generate

Generate AI-powered content using selected services.

Rate Limits

API requests are limited based on your plan tier. See Pricing for details.
