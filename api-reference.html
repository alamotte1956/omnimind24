API Reference

Authentication

All API requests require authentication using an API key. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Get your API key from your dashboard.

Base URL

https://api.omnimind24.com/v1

Campaigns

GET/campaigns

Retrieve all campaigns for your account.

Query Parameters

Parameter Type Required Description
page integer No Page number (default: 1)
limit integer No Results per page (default: 20, max: 100)
status string No Filter by status: active, paused, completed

Example Request

curl -X GET "https://api.omnimind24.com/v1/campaigns?limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"

Example Response

{
  "success": true,
  "data": [
    {
      "id": "camp_abc123",
      "name": "Summer Sale Campaign",
      "status": "active",
      "created_at": "2024-11-01T10:00:00Z",
      "metrics": {
        "impressions": 15000,
        "clicks": 450,
        "conversions": 23
      }
    }
  ],
  "pagination": {
    "page": 1,
    "total_pages": 5,
    "total_results": 47
  }
}

POST/campaigns

Create a new marketing campaign.

Request Body

Field Type Required Description
name string Yes Campaign name
type string Yes email, social, content, video
costar object Yes COSTAR framework parameters
ai_services array Yes AI models to use (gpt-4o, claude-3, etc.)

Example Request

curl -X POST "https://api.omnimind24.com/v1/campaigns" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Holiday Promotion",
    "type": "email",
    "costar": {
      "context": "E-commerce holiday sale",
      "objective": "Drive 20% increase in sales",
      "style": "Festive and urgent",
      "tone": "Exciting",
      "audience": "Existing customers 25-45",
      "response": "Shop now with discount code"
    },
    "ai_services": ["gpt-4o", "dall-e-3"]
  }'

GET/campaigns/:id

Get details of a specific campaign.

PUT/campaigns/:id

Update an existing campaign.

DELETE/campaigns/:id

Delete a campaign.

Content Generation

POST/generate/text

Generate AI-powered text content.

Request Body

{
  "model": "gpt-4o",
  "prompt": "Write an email subject line for a summer sale",
  "max_tokens": 100,
  "temperature": 0.7
}

POST/generate/image

Generate AI images.

Request Body

{
  "model": "dall-e-3",
  "prompt": "Professional marketing banner with gold accents",
  "size": "1024x1024",
  "quality": "hd"
}

Analytics

GET/analytics/campaigns/:id

Get detailed analytics for a campaign.

Response Fields

  • impressions - Total views
  • clicks - Total clicks
  • conversions - Total conversions
  • ctr - Click-through rate
  • roi - Return on investment

Rate Limits

  • Free Plan: 100 requests/hour
  • Pro Plan: 1,000 requests/hour
  • Enterprise Plan: Unlimited

Rate limit headers are included in all responses:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1699027200

Error Codes

Code Status Description
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error, contact support
